# 天猫BI数据 — 运营上传操作手册

## 📌 每日任务（12:00前完成）

从生意参谋下载4类数据文件，上传到腾讯云COS。

---

## 一、登录腾讯云COS控制台

1. 打开浏览器访问：**https://console.cloud.tencent.com/cos5/bucket**
2. 找到存储桶：**tmall-bi-data-v1-1430009310**
3. 点击进入存储桶 → 进入 **data/** 文件夹

你会看到4个子文件夹：

```
data/
  ├── 商品销售分析/     ← 销售数据放这里
  ├── 站内投放/         ← 广告投放数据放这里
  ├── 站内流量来源/     ← 流量数据放这里
  └── 商品退款分析/     ← 退款数据放这里
```

---

## 二、从生意参谋下载数据

### 📂 文件1：商品销售分析
- **路径**：生意参谋 → 数据 → 商品 → 商品销售分析
- **时间范围**：最近30天
- **格式**：Excel (.xlsx)
- **命名建议**：`商品销售分析_YYYYMMDD.xlsx`
- **上传到**：`data/商品销售分析/`

### 📂 文件2：站内投放（推广数据）
- **路径**：生意参谋 → 数据 → 推广 → 站内投放
- **时间范围**：最近14天
- **格式**：Excel (.xlsx)
- **命名建议**：`站内投放_YYYYMMDD.xlsx`
- **上传到**：`data/站内投放/`

### 📂 文件3：站内流量来源
- **路径**：生意参谋 → 数据 → 流量 → 站内流量来源
- **时间范围**：最近14天
- **格式**：Excel (.xlsx) 或 CSV
- **命名建议**：`站内流量来源_YYYYMMDD.xlsx`
- **上传到**：`data/站内流量来源/`

### 📂 文件4：商品退款分析
- **路径**：生意参谋 → 数据 → 商品 → 商品退款分析
- **时间范围**：最近14天
- **格式**：CSV 或 Excel
- **命名建议**：`商品退款分析_YYYYMMDD.xlsx`
- **上传到**：`data/商品退款分析/`

---

## 三、上传步骤（COS控制台）

1. 进入对应文件夹（如 `data/商品销售分析/`）
2. 点击右上角 **「上传文件」** 按钮
3. 选择本地已下载的文件
4. 等待上传完成（显示 ✅）
5. 重复以上步骤，上传其余3个文件夹的文件

---

## 四、⚠️ 注意事项

| 项目 | 要求 |
|------|------|
| **截止时间** | 每天中午 **12:00** 前 |
| **文件格式** | Excel (.xlsx) 或 CSV 均可 |
| **覆盖规则** | 新文件直接上传，会自动覆盖旧文件 |
| **缺文件怎么办** | 如果某天某个分类没有新数据，可以不上传，系统会用上一次的数据 |
| **文件名** | 随意命名，系统自动识别内容 |

---

## 五、上传完成后

无需其他操作！系统会在 **12:00 自动运行**：
- 从COS拉取最新文件
- 解析数据
- 更新仪表盘

📊 **仪表盘地址**：https://wg887k4tf4-lang.github.io/tmall-bi-dashboard/

---

## 六、遇到问题？

| 问题 | 解决方法 |
|------|---------|
| 登录不了腾讯云 | 联系大哥开通子账号权限 |
| 找不到存储桶 | 确认bucket名称：`tmall-bi-data-v1-1430009310` |
| 上传失败 | 检查文件大小（单文件不超过 5GB） |
| 仪表盘没更新 | 确认12:00后是否已上传 + 检查文件格式是否正确 |
