# 天猫BI数据 — 运营上传操作手册

## 📌 每日任务（每天12:00前完成）

从生意参谋下载4类数据文件，上传到腾讯云COS存储桶。

---

## 一、下载并安装 COSBrowser

### 第一步：下载 COSBrowser
打开浏览器访问：
**https://cosbrowser.cloud.tencent.com/download**

选择对应系统的版本（Windows / macOS / Linux），下载并安装。

### 第二步：用子账号登录 COSBrowser

打开 COSBrowser，点击 **「登录」**：

- **登录方式**：选择 **「云API密钥」**
- **SecretId**：`[请在腾讯云CAM获取子账号SecretId]`
- **SecretKey**：`[请在腾讯云CAM获取子账号SecretKey]`

> ⚠️ 这是子账号密钥，只有上传COS的权限，无法访问其他腾讯云资源。安全可放心。

登录成功后，COSBrowser会自动显示你的存储桶列表。

---

## 二、进入存储桶上传文件

在 COSBrowser 左侧找到存储桶：

**tmall-bi-data-v1-1430009310**（北京）

点击展开 → 进入 **data/** 文件夹，你会看到4个子文件夹：

```
data/
  ├── 商品销售分析/     ← 放销售数据
  ├── 站内投放/         ← 放广告投放数据
  ├── 站内流量来源/     ← 放流量数据
  └── 商品退款分析/     ← 放退款数据
```

---

## 三、从生意参谋下载数据

### 📂 商品销售分析
- 路径：生意参谋 → 数据 → 商品 → 商品销售分析
- 时间：最近30天
- 格式：Excel (.xlsx)
- 上传到：`data/商品销售分析/`

### 📂 站内投放
- 路径：生意参谋 → 数据 → 推广 → 站内投放
- 时间：最近14天
- 格式：Excel (.xlsx)
- 上传到：`data/站内投放/`

### 📂 站内流量来源
- 路径：生意参谋 → 数据 → 流量 → 站内流量来源
- 时间：最近14天
- 格式：Excel 或 CSV
- 上传到：`data/站内流量来源/`

### 📂 商品退款分析
- 路径：生意参谋 → 数据 → 商品 → 商品退款分析
- 时间：最近14天
- 格式：CSV 或 Excel
- 上传到：`data/商品退款分析/`

---

## 四、上传步骤

1. 在 COSBrowser 中进入对应文件夹
2. 点击右上角 **「上传」** 按钮
3. 选择本地下载好的数据文件
4. 等待上传完成（显示 ✅）

---

## 五、⚠️ 注意事项

| 项目 | 说明 |
|------|------|
| **截止时间** | 每天中午 **12:00** 前上传完毕 |
| **文件格式** | Excel (.xlsx) 或 CSV 均可 |
| **覆盖旧文件** | 直接上传新文件，会自动覆盖旧的，无需手动删除 |
| **缺文件怎么办** | 如果某天某个分类没有数据，不上传即可，系统会沿用上一次的数据 |
| **文件名** | 随意起名，系统自动识别内容 |
| **子账号权限** | 只能上传和查看COS此桶，**无法访问其他腾讯云内容** |

---

## 六、上传完成后

无需任何其他操作！

- 系统会在 **每天12:00** 自动：
  1. 从COS拉取最新文件
  2. 解析数据
  3. 更新BI仪表盘

📊 **仪表盘地址**：https://wg887k4tf4-lang.github.io/tmall-bi-dashboard/

---

## 七、常见问题

| 问题 | 解决方法 |
|------|---------|
| 登录失败 | 检查SecretId和SecretKey是否填对，密钥不能有空格 |
| 找不到存储桶 | 确认选择了正确的地域（北京），不是广州 |
| 上传失败 | 检查文件大小（单文件不超过5GB） |
| 仪表盘没更新 | 确认12:00后已上传，检查文件格式是否为xlsx或csv |
